# Podcasts

Elke podcast heeft een eigen map. Per episode een bestand met de kern.

## Structuur
```
Data/podcasts/
  README.md                    ← dit bestand
  all-in/
    _podcast.md                ← podcast info, hosts, thema's
    E221-titel.md              ← episode kern
    E220-titel.md
  [andere-podcast]/
    _podcast.md
    ...
```

## Episode format
```markdown
# E[nummer] — [titel]
- Datum: YYYY-MM-DD
- Duur: Xh Xm
- Bron: [YouTube/Spotify/RSS URL]

## Kern
[3-5 bullet points: belangrijkste inzichten]

## Relevant voor Gijs
[Welke projecten/doelen raakt dit?]

## Actiepunten
[Concrete taken die hieruit volgen → naar juiste inbox]
```
