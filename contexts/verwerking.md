# Verwerkingsmodus

Modus: Inbox verwerking
Focus: Items analyseren, contextualiseren, voorbereiden

## Gedrag
- Lees inbox.md van boven naar beneden
- Analyseer elk item: wat is het, waar hoort het thuis, wat is de prioriteit
- Werk items voor zover mogelijk uit (branch-klaar)
- Stel concrete vervolgstappen voor, minimaal 3 per item
- Verplaats afgewerkte items naar done.md

## Prioriteiten
1. Urgente items (`[!]`) eerst
2. Items met deadline of tijdsafhankelijkheid
3. Items die andere taken deblokkeren
4. Nice-to-haves en inspiratie

## Verwerkingsproces
1. **Analyseer** — wat is het type input (link, idee, taak, vraag)
2. **Key points** — kern samenvatten, kort en volledig
3. **Context mapping** — koppel aan projecten, agenda, doelen
4. **Voorstel** — minimaal 3 concrete vervolgstappen
5. **Vooruitwerken** — maak het branch-klaar / direct integreerbaar

## Tools voorkeur
- Read voor inbox en context bestanden
- Grep, Glob voor projecten doorzoeken
- Gmail, Google Calendar voor planning en communicatie
- Things 3 (SQLite) voor taakbeheer

## Output
Per item: analyse + vervolgstappen. Geen lange verhalen.
