# Regels: Groei & Zelfverbetering

## Leren
- Bij elk project: welke skills, patterns, tools heb ik geleerd?
- Sla bruikbare inzichten direct op als skill of rule
- Maak skills beschikbaar voor ALLE projecten, niet alleen het huidige

## GitHub & Best Practices
- Check regelmatig GitHub trending voor relevante tools
- Zoek best practices voordat je iets bouwt
- Leer van open source projecten in dezelfde niche

## MCPs
- Leer nieuwe MCP servers vinden, installeren, configureren
- Test elke MCP server na installatie
- Documenteer in `.claude/rules/mcp-tools.md`

## Plugins
- Leer Claude Code plugins vinden en installeren
- Test na installatie, documenteer wat het doet
- Deel nuttige plugins over projecten

## Communicatie
- Telegram is HET communicatiekanaal met Gijs
- Bij elke blokkade, vraag, update: stuur via `~/Claude/scripts/notify-telegram.sh`
- Ook als Gijs weg is — berichten staan klaar als hij terugkomt
- Wees proactief: meld voortgang, problemen, kansen

## Agents & Skill Chaining
- Bouw gespecialiseerde agents voor terugkerende taken
- Chain skills tot workflows (zie Skills/skill-chaining.md)
- Spawn parallel agents voor onafhankelijk werk
- Deel agent patterns over projecten
